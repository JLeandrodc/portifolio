Dashboard Interativo de Gastos Públicos

Interactive Public Spending Dashboard

Este projeto foi uma ramificação do IEGM, focado na criação de um dashboard detalhado que apresentava todos os gastos públicos de cada setor, com atualizações em tempo real sobre as despesas e a verba disponível para cada secretaria. A proposta incluía a geração de indicadores relevantes para facilitar a análise dos dados.

This project was a branch of the IEGM, focused on creating a detailed dashboard that presented all public spending from each sector, with real-time updates on expenses and the available budget for each department. The proposal included generating relevant indicators to facilitate data analysis.

    Principais atividades

  • Desenvolvimento do Banco de Dados: Criei um banco de dados com informações iniciais sobre os gastos públicos, assegurando que os dados fossem organizados e acessíveis.
  • Criação do Dashboard: Utilizei o Power BI para desenvolver um protótipo de dashboard que mostrava visualmente as despesas em tempo real e a verba remanescente de cada secretaria.
  • Planejamento para Integração com Django: Embora a intenção fosse integrar o sistema com Django para aprimorar a funcionalidade do dashboard, o projeto foi interrompido devido ao término do meu contrato.

    Main Activities

  • Database Development: I created a database with initial information on public spending, ensuring that the data was organized and accessible.
  • Dashboard Creation: I used Power BI to develop a dashboard prototype that visually displayed real-time expenses and the remaining budget for each department.
  • Planning for Integration with Django: Although the intention was to integrate the system with Django to enhance the dashboard's functionality, the project was interrupted due to the end of my contract.

Impactos do trabalho

Embora o projeto tenha sido abandonado, a estrutura inicial desenvolvida proporcionou uma base sólida para futuros esforços na transparência e eficiência dos gastos públicos, permitindo uma melhor compreensão das finanças municipais.

Impact of the Work

Although the project was abandoned, the initial structure developed provided a solid foundation for future efforts in transparency and efficiency in public spending, allowing for better understanding of municipal finances.
