IEGM

IEGM

O IEGM (Índice de Governança Municipal) avalia a qualidade da governança nos municípios brasileiros, considerando transparência e eficiência. Seu objetivo é identificar áreas de melhoria e promover uma gestão pública mais responsável.

The IEGM (Municipal Governance Index) evaluates the quality of governance in Brazilian municipalities, considering transparency, citizen participation, and efficiency. Its goal is to identify areas for improvement and promote more responsible public management.

Projeto DIA - Dashboard de Interação de Dívidas Ativas

DIA Project - Active Debt Interaction Dashboard

O Projeto DIA visou criar uma ferramenta visual que facilite a gestão e análise das dívidas, proporcionando números e dados claros. O intuito é aumentar a transparência e a eficiência na administração financeira.

The DIA Project aimed to create a visual tool to facilitate the management and analysis of debts, providing clear numbers and data. The goal is to increase transparency and efficiency in financial administration.

Projeto DGP - Dashboard de Gastos Públicos

DGP Project - Public Spending Dashboard

O Projeto DGP visou criar uma ferramenta visual que facilitasse a gestão e a análise dos gastos públicos, apresentando dados detalhados sobre despesas de todos os setores. O intuito era proporcionar atualizações em tempo real sobre os gastos e a verba disponível para cada secretaria.

The DGP Project aimed to create a visual tool to facilitate the management and analysis of public spending, presenting detailed data on expenses from all sectors. The goal was to provide real-time updates on spending and the funds available for each department.
